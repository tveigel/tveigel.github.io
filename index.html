4. RESULTS

We conducted repeated measures ANOVAs with Speed-Accuracy Tradeoff (SAT: Conservative, Moderate, Risky), Uncertainty Communication (Yes, No), and User Decision (Yes, No) as within-subjects factors. Effect sizes are reported as generalized eta-squared (η²G).

4.1 User Frustration

Frustration with Robot Performance. SAT significantly affected user frustration about robot performance, F(2, 46) = 73.15, p < .001, η²G = .550. Frustration increased from conservative (M = 2.09, SD = 1.34) to moderate (M = 3.79, SD = 1.60) to risky conditions (M = 5.04, SD = 1.81). All pairwise comparisons were significant (p < .001). A significant SAT × User Decision interaction emerged, F(2, 46) = 7.45, p = .002, η²G = .051.

Frustration with Robot Decisions. SAT significantly affected frustration about robot decisions, F(2, 46) = 52.54, p < .001, η²G = .413, with conservative (M = 1.99, SD = 1.25), moderate (M = 3.23, SD = 1.66), and risky conditions (M = 4.04, SD = 2.07). User Decision also showed a main effect, F(1, 23) = 5.20, p = .032, η²G = .074. A significant SAT × User Decision interaction was observed, F(2, 46) = 10.88, p < .001, η²G = .073.

4.2 Workload (NASA TLX)

Overall Workload. SAT significantly affected overall workload, F(2, 46) = 27.74, p < .001, η²G = .110. The SAT × User Decision interaction was significant, F(2, 46) = 11.48, p < .001, η²G = .014.

TLX Subscales. SAT significantly affected mental demand (F(2, 46) = 7.45, p = .002, η²G = .016), physical demand (F(2, 46) = 17.43, p < .001, η²G = .072), temporal demand (F(2, 46) = 9.27, p < .001, η²G = .058), effort (F(2, 46) = 22.80, p < .001, η²G = .126), and frustration (F(2, 46) = 22.01, p < .001, η²G = .151). User Decision showed a main effect on perceived performance, F(1, 23) = 8.88, p = .007, η²G = .005.

4.3 Perceived Utility

SAT significantly affected perceived utility, F(2, 46) = 93.98, p < .001, η²G = .563. Utility decreased from conservative (M = 3.26, SD = 0.66) to moderate (M = 2.46, SD = 0.74) to risky conditions (M = 1.74, SD = 0.71). All pairwise comparisons were significant (p < .001). A significant SAT × User Decision interaction was found, F(2, 46) = 9.08, p < .001, η²G = .069.

4.4 Perceived Intelligence

SAT significantly affected perceived intelligence, F(2, 46) = 53.95, p < .001, η²G = .463. Intelligence ratings decreased from conservative (M = 3.76, SD = 0.77) to moderate (M = 3.10, SD = 0.68) to risky conditions (M = 2.49, SD = 0.87). User Decision showed a main effect, F(1, 23) = 12.52, p = .002, η²G = .096. The SAT × User Decision interaction was significant, F(2, 46) = 12.14, p < .001, η²G = .070.

4.5 Summary

The SAT manipulation produced robust effects across all dependent measures, with large effect sizes for frustration (η²G = .41–.55), utility (η²G = .56), and perceived intelligence (η²G = .46). Riskier robot behavior consistently led to higher frustration, greater workload, lower perceived utility, and reduced perceived intelligence. Notably, Uncertainty Communication showed no significant main effects, while User Decision and SAT × User Decision interactions emerged for several measures, suggesting that user agency moderates the impact of SAT on user experience.
